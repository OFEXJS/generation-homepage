# 管理个人日志

一个基于现代前端技术构建的个人日志打包工具，快速创建、管理和分享个人日志内容。

## 项目概述

本项目是个人的日志打包工具，提供直观的界面和流畅的用户体验，轻松记录生活点滴和思考感悟。

## 技术栈

- **前端框架**: React
- **编程语言**: TypeScript
- **构建工具**: Vite
- **样式方案**: 原生 CSS

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 使用场景

- 日常记录生活感悟
- 工作学习笔记整理
- 灵感创意捕捉
- 个人成长追踪
